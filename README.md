# remote-data-admin-project
Deskripsi
Project ini di buat untuk mengotomisasi proses import data CSV ke PostgreSQL menggunakan n8n serta menampilkan hasil analisis melalui Grafana Dashboard. 

Workflow
CSV dataset --> n8n --> PostgreSQL --> Grafana --> Telegram Notification

Tools Yang digunakan
- PostgreSQL
- n8n
- Grafana
- Telegram Bot
- Linux

Fitur
- Membaca Dataset CSV secara otomatis
- Memproses data menggunakan n8n
- Menampilkan Dashboard analisis di Grafana
- Mengirim Notifikasi otomatis ke Telegram

Hasil Project
- Data berhasil diimport ke PostgreSQL
- Dashboard Grafana berhasil dibuat
- Notifikasi Telegram berhasil dikirim otomatis
